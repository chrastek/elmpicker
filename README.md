# Elmpicker

## About

Simple elm SPA boilerplate with
* Dev server with live reloading, HMR
* Elm-css
* Bundling and minification for deployment

## Install

```
git clone https://github.com/dominikchrastek/elmpicker
cd elmpicker
```

Reinstall project
```
yarn run reinstall
```
Serve locally
```
yarn start
```
Build
```
yarn run build
```
